    \section{进程与线程}

    \begin{enumerate}
        \item 在多道程序环境下，允许多个程序并发执行，此时它们将失去封闭性，并具有间断性即不可再现性的特征，故引入进程以便更好地描述和控制程序的并发执行，实现操作系统的并发性和共享性
        \item 进程（Process）
        \begin{itemize}
            \item 正在执行程序的实例
        \end{itemize}
        \item 进程实体（进程映像） = 程序段 + 相关数据段 + PCB。反映了进程在某一时刻的状态
        \item PCB（Process Control Block，进程控制块）：描述进程的基本情况和运行状态，进而控制和管理进程
    \end{enumerate}
